Java Task 1 – Hello World & Command Line Arguments

This repository contains my solution for Task 1 of the Java Developer Internship. The purpose of this task is to set up the Java environment, write a basic Java program, compile it, run it, and use command line arguments.

Task Overview

The task includes:

Installing and verifying Java (JDK)

Writing a simple Java program

Compiling the program using javac

Running the program using java

Passing and printing command line arguments

Following basic coding and commenting standards

Program Description

The Java program prints:

A greeting message: Hello World

All command line arguments passed during execution

Source Code

File: HelloWorld.java

The program uses:

main(String[] args) to receive inputs from the command line

A for loop to print each argument on a new line

How to Run the Program

Open terminal or command prompt in the project directory.

Step 1: Compile the program
javac HelloWorld.java

Step 2: Run without arguments
java HelloWorld

Step 3: Run with arguments
java HelloWorld Welcome To Java Programming

Sample Output

When running:

java HelloWorld Welcome To Java Programming


The output will be:
